# 个人知识问答聊天机器人 Demo

本项目是一个面向个人知识问答和通用文本聊天场景的大模型应用原型。

## 技术栈

项目使用 Python、FastAPI、SQLite、大模型 API、RAG、向量检索和 Prompt Engineering。后端提供 REST API，SQLite 保存会话、消息、关键词命中、知识库来源和模型回答。

## 对话流程

系统收到用户输入后，会先保存用户消息，然后进行意图识别。意图包括关键词回复、指令工具、知识库问答和普通聊天。不同意图会路由到不同服务，最后保存助手回答并返回接口响应。

## RAG 知识库

知识库支持导入 txt 和 markdown 文件。系统会清洗文本、切分片段、生成向量并进行相似度检索。检索到的片段会作为上下文注入大模型 Prompt，回答中保留资料来源。

## 工具调用

Demo 支持时间查询和天气查询。时间查询使用本地函数，天气查询在没有 API Key 时返回 mock 数据，方便演示。配置天气 API Key 后可接入真实服务。

## 工程价值

项目重点验证大模型在垂直知识问答、工具调用和对话流程管理中的工程落地能力，适合作为简历项目、面试演示和二次扩展基础。
